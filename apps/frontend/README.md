# @panchnity/frontend

Production-ready Next.js 16 frontend for the Hold Yourself monorepo.

## Getting Started

1.  **Install dependencies** (from root):

    ```bash
    pnpm install
    ```

2.  **Environment Variables**:
    Copy `.env.example` to `.env` and fill in the values.

    ```bash
    cp .env.example .env
    ```

3.  **Run development server**:
    ```bash
    pnpm dev
    ```

## Features

- **Next.js 16 (App Router)**: Leveraging the latest performance improvements.
- **Tailwind CSS**: Utility-first styling.
- **First-class SEO**: Built-in Metadata API with dynamic robots and sitemap.
- **Strict Environment Validation**: Fails fast if required variables are missing.
- **Turborepo Compatible**: Optimized for monorepo orchestration.
